# fissible

Developer tools built for the terminal — bash-first, dependency-light, and designed to stay out of your way.

## Projects

### Bash tooling

| Project | Description |
|---|---|
| [seed](https://github.com/fissible/seed) | Fake data generator — 42 generators, 4 output formats, MCP server for Claude Code |
| [shellframe](https://github.com/fissible/shellframe) | Bash TUI primitives: alternate screen, raw input, ANSI-aware column padding |
| [shellql](https://github.com/fissible/shellql) | Terminal SQLite workbench built on shellframe |
| [ptyunit](https://github.com/fissible/ptyunit) | PTY-driven test framework for bash and TUI applications |
| [macbin](https://github.com/fissible/macbin) | Personal productivity tools for macOS |
| [guit](https://github.com/fissible/guit) | Terminal UI git client — visual staging, merging, history, branch management |

### PHP
